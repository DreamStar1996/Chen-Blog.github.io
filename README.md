## 🛠 技术栈 (Tech Stack)

本博客系统追求极致的性能与可持续维护性，技术架构如下：

- **核心框架**：[Astro](https://astro.build/) + MDX
  - 用于构建极简、高性能的静态站点，确保五年后依然能够顺利运行。
- **3D 引擎**：[Three.js](https://threejs.org/) (r128)
  - 驱动首页的“曲率引擎”跃迁动画，模拟星点加速、星轨拉长、曲率隧道等科幻视觉效果。
- **样式设计**：CSS3 + Google Fonts
  - **配色**：深空黑 (`#050510`) 搭配星云紫、青、金点缀。
  - **字体**：`Noto Serif SC` (衬线正文) + `Cormorant Garamond` (标题) + `JetBrains Mono` (代码/技术流)。
- **部署环境**：静态文件托管 (HTML/CSS/JS)
