# 更新说明 · 本次迭代

> 目标：① 整体放大字号；② 把「观星志」的全部组件迁移齐全；③ 补充若干 canvas 特效，为后续转 Hexo 主题做准备。
> 所有改动均沿用既有主题变量，自动随「日 / 夜」切换；曲率引擎（warp）跃迁动画原样保留，未作改动。

## ① 字号整体放大（可读性）
- **基准字号**上调：`html { font-size }` 现为 `clamp(18.5px, 0.52vw + 16.6px, 21px)`（原约 16.5–19px）。
  想整体调大 / 调小，**只改 `css/style.css` 里这一处的三个数字**即可（迁移 Hexo 后可挂到 `_config` 字段）。
- **集中调参块**（`css/style.css` 末尾「字号微调」一节）统一上调了：眉标 / 徽标 / 元信息、正文、摘要、卡片标题、面板标题等所有偏小的字号，并覆盖了迁移来的组件。
- 手机端（≤560px）做了紧凑化，避免放大后顶栏 logo / 徽标换行；正文在窄屏略收，杜绝横向溢出。

## ② 观星志组件 · 补齐迁移（均在「跃迁抵达」后的博客界面内）
- **HERO 读出条**：坐标 · 天气 · 月相（巴黎 48.86°N / 2.35°E），随日 / 夜切换文案。
- **品牌星徽**：罗盘八角星 SVG，置于博客 HERO 顶部。
- **今夜观测面板**：补上「太阳 / 月亮」日夜图标与文案切换（白天显示太阳 + 今日天气）。
- **通用按钮** `.btn / .btn-gold / .btn-ghost`：HERO 增加「浏览手记 / 看看友链」两枚行动按钮。
- **富页脚**：把原先单行页脚换成四栏页脚（品牌 + 星座 / 导航 / 关注 + 底部坐标行）。

## ③ canvas 特效 · 新增（Hexo 预备）
- **背景引擎从 3 种扩展为 5 种**（左下角「✦ 背景」胶囊循环切换，记忆上次选择）：
  `星空 COSMOS → 星轨 STARTRAIL → 流星 METEOR → 花瓣 PETALS → 萤火 FIREFLY`
  - **花瓣 PETALS**：主题配色（金 / 玫 / 冰 / 白）的程序化花瓣，斜向飘落 + 自旋，无需任何外部图片；白天亦保留可见度。
  - **萤火 FIREFLY**：带柔光晕的金 / 冰光点，缓慢游走并呼吸明灭；白昼自动隐去。
  - 引擎自带「Hexo 迁移提示」：未来可由 `_config` 的一个字段决定默认背景模式，其余照搬。
- **友人帐 · 友链墙**：Hexo 经典页面，复用星云卡片美学（头像 + 站名 + 简介，6 张示例卡，按分类着色）。
- **阅读进度条**（顶部）+ **返回顶部**按钮（右下）：仅在博客界面启用，随滚动显隐。

1. **博客浏览页加宽**。内容区从原先 `max-width:1180px` + `8vw` 留白（笔记本上实际仅约 1210px）统一放宽到 **1320px**，两侧留白改为 `clamp(24px, 4vw, 64px)`——宽屏更舒展、窄屏依旧安全。调宽 / 调窄只需改 `css/style.css` 里的 `--blog-w` 与 `--blog-pad` 两个变量。

## 四次调整（根据反馈）
1. **中文字体改为「楷体」**。全站中文统一换成 **霞鹜文楷 LXGW WenKai**（开源 Kai 体，含书法笔意），通过 jsDelivr CDN 加载；降级链为 系统楷体（楷体 / KaiTi / STKaiti / 标楷体）→ serif，离线或加载前即以系统楷体显示，`font-display:swap` 平滑切换。拉丁标题仍用 Cormorant Garamond，与楷体相衬。字体定义集中在 `--kai` 变量，便于统一替换。配合楷体笔画特点，正文行距与字重做了微调。
2. **整体字号再小一码**。基准从 `clamp(18.5px, 0.52vw + 16.6px, 21px)` 降到 `clamp(17.5px, 0.5vw + 15.7px, 20px)`（手机 17px）。仍是 `css/style.css` 顶部一处变量，便于继续微调。
3. **响应式排版加强（不止「不溢出」，而是逐档适配）**。在原有零散断点之上补齐统一的三档体系，置于文件末尾确保优先级：
   - **≤1100px**（窄桌面 / 横屏平板）：星座卡 3→2 列、精选大卡与今夜面板提前收成单列、友链与页脚转双列。
   - **≤820px**（竖屏平板）：标题阶梯整体下调、双列信息区转单列、星云/友链转单列、区段竖向留白收紧、天象历隐藏标签列。
   - **≤560px / ≤400px**（手机）：HERO 按钮纵向整宽、区段标题纵向排布避免「更多」挤压、公告卡/时间线/今夜/友链/订阅内边距逐级收紧。
   - **触摸设备**（`hover:none`）：禁用 hover 位移类动效，避免移动端「悬停卡住」。
   实测桌面 1440 / 平板 834 / 手机 390 三种宽度、公告页与博客页均无横向溢出。
